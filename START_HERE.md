## 🆘 Common Issues & Solutions

**Q: App won't start?**
```bash
rm -rf node_modules package-lock.json
npm install
npm run dev
```

**Q: Port 3000 already in use?**
```bash
npm run dev -- --port 3001
```

**Q: Styles look weird?**
- Hard refresh: `Ctrl + Shift + R`
- Check if `global.css` is in `src/main.jsx`

**Q: Demo login doesn't work?**
- Check email spelling exactly (case sensitive for email)
- Password is also case sensitive

**Q: Missing pages?**
- Make sure all routes are in `src/App.jsx`
- Check spelling of route paths
